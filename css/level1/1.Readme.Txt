Css is used to give styling to the website .

There are Three ways to give css 
    1) Inine css ==> using css inside a tag
    2) Internal css ==> adding css in style tag (style tag should be placed inside head tag) , we use selector in this css
    3) External css  ==> writtng css to a file and linking that file to the html file by link tag


Note :-  more prefence is given to the inline css
